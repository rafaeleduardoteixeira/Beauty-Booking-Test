## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

Start both Next.js and JSON Server:

```bash
npm run dev:all
```

Or run them separately:

```bash
# Terminal 1 - Next.js (port 3000)
npm run dev

# Terminal 2 - JSON Server (port 3001)
npm run server
```
